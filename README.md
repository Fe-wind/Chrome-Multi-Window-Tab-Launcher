# なにやつ
- Chrome の拡張機能

# やれること
- Chrome ウィンドウ内で開いている複数のタブをすべてウィンドウにバラす

# やりかた
- ブラウザ上（なにもないところ）で右クリック→「すべてのタブを別々の新しいウィンドウで開く」→全てのタブが別々の新しいウィンドウで開く
